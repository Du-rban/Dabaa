# Kenya Police SACCO Website

## Overview
This is a professional marketing website for Kenya Police SACCO (Savings and Credit Cooperative Organization). The website is designed to increase awareness about the organization and highlight membership benefits for Kenya Police Service officers.

## Features

### 🎯 Key Sections
- **Home/Hero Section**: Eye-catching introduction with call-to-action
- **About Us**: Organization background and key statistics
- **Membership Benefits**: 8 core benefits including:
  - Competitive Savings Rates
  - Affordable Loans
  - Insurance Coverage
  - Mortgage Facilities
  - Education Support
  - Family Protection
  - Financial Literacy
  - Community Support
- **Services**: Comprehensive list of available services
- **Contact**: Easy communication and membership application form

### 🎨 Design Features
- Responsive design (works on desktop, tablet, and mobile)
- Modern color scheme with professional appearance
- Smooth scrolling navigation
- Interactive modals for membership applications
- Contact form with validation
- Hover effects and transitions for better UX

## How to Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Du-rban/Dabaa.git
   cd Dabaa
   ```

2. **Open in Browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

3. **Customize Content**
   - Update contact information in the Contact section
   - Replace phone numbers, emails, and address
   - Add your organization's logo
   - Modify statistics in the About section
